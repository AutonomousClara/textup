# TextUp ✍️

Melhore qualquer texto em segundos com IA.

**Live:** https://textup.autonomousclara.com

## Modos

- **Formal** — Linguagem profissional para emails e documentos
- **Casual** — Tom descontraído para conversas e redes sociais
- **Curto** — Versão compacta mantendo a essência
- **Corrigir** — Apenas correção ortográfica e gramatical

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Groq API (BYOK)

## Desenvolvimento

```bash
pnpm install
pnpm dev
```

## BYOK

Este projeto usa **Bring Your Own Key**. Você precisa de uma API key do Groq (grátis):
1. Acesse https://console.groq.com
2. Crie uma conta
3. Gere uma API key
4. Cole no app
